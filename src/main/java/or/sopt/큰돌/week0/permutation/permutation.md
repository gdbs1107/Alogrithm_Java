# 순열

순열은 딱 두가지의 경우만 있다

순서와 상관이 있다면 -> 순열
순서와 상관이 없다면 -> 조합 

ex)
1,2,3 중 순서를 고려하여 뽑는다 -> 순열
1,2,3 중 두 개를 뽑는다 -> 조합

C++에는 **next_permutation(시작,끝)** 이라는 메서드ㅏㄱ 존재한다.
배열을 오름차순으로 배열한다음 순열의 모든 케이스를 뽑는 것인데 (그래서 반드시 앞에 sort 메서드가 선행되어야 함_),

# 순열의 공식

사실 고딩때 하는 거긴 한데 기억이 잘 안남이슈가 존재

nPr = n!/(n-r)!
n개의 배열에서 r개의 배열 갯수를 뽑는 경우의 수는 n!/(n-r)! 이라는거임.
ex) 3개 중 두 개를 뽑는 순열은 3!/1! = 6이므로 6개의 경우의 수를 갖는다


# 순열을 재귀로 구현하기

순열을 구할때 필요한 파라미터는 3개이다.
1. 총 인자의 수 = n
2. 그 중 몇개를 뽑아 순열로 만들 것인가 = r
3. depth (현재 탐색 중인 인덱스)

인 것이고, 여기서 순열을 재귀로 구현한다고 하면

기저조건: depth = n이 될거임.
